# Chimera — AI Evidence Intelligence Platform

Chimera is a premium, high-end AI-powered SaaS product and full-stack application designed for evidence extraction, contradiction detection, and timeline analysis. It features a futuristic Command Center interface with interactive data visualization, real-time analytics, and visual synthesis pipelines.

## Key Features

1. **Operations Command Center**
   - **System Dashboard**: Central operations panel to view case summaries, ingestion rates, and system alerts.
   - **Multimodal Ingestion**: Upload unstructured files (PDFs, transcripts, audio) to extract semantic records.
   - **Lattice Node Graph**: Interactive network mapping of entities, locations, and contradictions.
   - **Synaptic Timeline**: Chronological sequence mapping of events extracted across multiple documents.
   - **AI Transmission Chat**: Query the unified knowledge base with citation-grounded conversational answers.
   - **Contradiction Analyzer**: Cross-examine multi-source materials to flag contradictions and inconsistencies.
   - **Neural Analytics**: Visual token distributions, extraction confidence metrics, and system health monitors.
   - **System Settings**: Manage API configurations, custom taxonomies, and model selection.

2. **Synthesis Pipeline**
   - High-fidelity visual workflow representing data progression:
     1. **Document Ingestion** (PDF / Audio parsing)
     2. **Entity Extraction** (NER / Location mapping)
     3. **Contradiction Analysis** (Discrepancy flags)
     4. **Evidence Synthesis** (Verdict / Summary generation)

3. **Premium Cinematic Interface**
   - **3D Living Brain Network**: A custom Three.js WebGL visualizer rendering interactive neural nodes, pulsating connecting lines, and orbiting ring particles reacting dynamically to system states.
   - **Floating Interactive Bubbles**: Dynamic canvas-based drifting glass bubbles with parallax depth, horizontal sine-wave wobbling, and smooth mouse cursor repulsion.
   - **Responsive Navigation & Audio Drone**: Full accessibility controls, background neural hum drone, and click feedback sounds.

## Recent Progress & Updates

- **Generic Renaming**: Replaced terminology from *"Judges"* to a generic **Operations Console** and **Operations Command Center** to make the platform accessible to all users.
- **Background Bubble Physics**: Integrated moving glass bubbles onto the landing page background canvas reacting to mouse positioning.
- **Code Optimization**: Rebuilt and tested the production package with fully type-safe React Router configurations.

## Technology Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Routing**: TanStack Router
- **Interactive Visuals**: Three.js, HTML5 Canvas 2D
- **Backend / Integration**: Lovable & Supabase client integration
